title: += オペレータ
---

整数に使用するオペレータでよく使われる`+=`オペレータを紹介します。 

まずは下のコードを見てみてください。 

```
>>> a = 1
>>> a += 2
>>> a
3
```

1行目で「a」という変数に「1」という整数を代入しています。 

2行目で、`+=`オペレータを使って、「a」という変数に「a」に「2」を加えた整数を代入しています。 

`+=`オペレータは、ある変数に、その変数を使った足し算の結果を代入する場合に使います。 

```
a += 1
```

はコードとしては、 

```
a = a + 1
```

と同じことです。 

* `-=`、 `*=`、 `/=`

も、それぞれ引き算、掛け算、割り算で同じ動作をします。
